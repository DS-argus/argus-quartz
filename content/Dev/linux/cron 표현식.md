### 크론(cron) 표현식이란?  
- 일정 기반 자동화에 쓰는 시간 예약 포맷
- 주로 리눅스 crontab, Apache Airflow, Kubernetes CronJob, Jenkins 등 워크플로/스케줄러에서 작업 실행 시간을 지정할 때 사용
- Airflow, k8s 등에서는 실행 명령어 대신 **5개 시간 필드만** 사용
- 흔히 “cron schedule”, “cron expression”이라고 부름

---

### 기본 구조 : '분 시 일 월 요일'
- 각 필드는 아래 표 기준으로 순서대로 입력함

| 필드  |      허용 범위      |             의미             |
| :-: | :-------------: | :------------------------: |
|  분  |      0-59       |           minute           |
|  시  |      0-23       |            hour            |
|  일  |      1-31       |        day of month        |
|  월  | 1-12 or JAN-DEC |           month            |
| 요일  | 0-7 or SUN-SAT  | day of week (0과 7은 Sunday) |

##### 특수 기호  
- `*`  : 모든 값  
- `,`  : 여러 값 지정  
- `-`  : 범위 지정  
- `/`  : 주기 지정 (간격)  
- `?`  : 일과 요일 필드 중 하나를 무시할 때 사용 (Quartz 등 확장)[^1]  
- `L`  : 마지막 날 혹은 마지막 요일 (확장)  
- `#`  : 월요일#3 → 셋째 월요일처럼 n번째 요일 지정 (확장)


##### 평가 방식과 주의점  
- 5개 시간 필드는 왼쪽에서 오른쪽 순으로 평가
- “일”과 “요일”이 모두 지정되면 OR 관계로 동작 (둘 중 하나만 맞아도 실행)  
- 시스템/도구별 확장 기호 지원 여부 다를 수 있으니 공식 문서 참고해야 함

### 예시 모음
| 목적             | 표현                  | 설명                 |
| -------------- | ------------------- | ------------------ |
| 매 분마다          | `* * * * *`         | 1분마다 실행            |
| 매일 03:15       | `15 3 * * *`        | 매일 03시 15분         |
| 평일 18시 정각      | `0 18 * * 1-5`      | 월~금 18:00          |
| 주말 자정          | `0 0 * * 6,0`       | 토 00:00, 일 00:00   |
| 10분마다          | `*/10 * * * *`      | 10분 간격             |
| 매월 1일 02:00    | `0 2 1 * *`         | 매달 1일 02:00        |
| 분기마다 첫째날 04:30 | `30 4 1 1,4,7,10 *` | 1·4·7·10월 1일 04:30 |
| 매년 12-31 23:59 | `59 23 31 12 *`     | 연말 자정 1분 전         |
| 매주 월요일 09:00   | `0 9 * * 1`         | 월 09:00            |
| 특정 요일 n번째      | `0 8 ? * MON#2`     | 둘째 월요일 08:00 (확장)  |

[^1]: 확장이 붙은 것은 표준 리눅스 crontab에는 없는 기능
